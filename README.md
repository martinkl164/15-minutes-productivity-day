# 15-Min Productivity

Plan your day in 15-minute blocks. Set your work window, run a timer per slot, and log what you did in 1–2 sentences. Mobile-first, no account required.

**Live app:** [https://martinkl164.github.io/15-minutes-productivity-day/](https://martinkl164.github.io/15-minutes-productivity-day/)

---

## Features

- **Day view** — 15‑min slots for the selected day; start/stop timer and add a note per slot
- **Week view** — Summary of the week; tap a day to open the day view
- **Month view** — Calendar with indicators for days that have notes
- **Current time line** — Visual marker for “now” in the day (today only)
- **Themes** — Dark, Light, Sepia, Ocean, and Dashboard; configurable in Settings
- **Locale** — Time and date format follow your browser language
- **Offline** — Data stored in the browser (localStorage); no backend

## Deploy to GitHub Pages

1. Push this repo to GitHub.
2. In the repo: **Settings → Pages**.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. Push to the `master` branch (or trigger the workflow from the **Actions** tab).

The workflow in [.github/workflows/deploy-pages.yml](.github/workflows/deploy-pages.yml) builds the app and deploys it. The site will be available at:

**https://martinkl164.github.io/15-minutes-productivity-day/**

(Replace the username and repo name if you use a different repository.)

Core 15-Minute Productivity Techniques
The 15-Minute Rule (Overcoming Inertia): When feeling overwhelmed, commit to working on a task for just 15 minutes. This reduces pressure and often causes the motivation to "snowball" into longer productive sessions.
15-Minute Time Tracking (Alex Hormozi Method): Track your entire day in 15-minute increments (e.g., in a notepad or app) to see exactly where time is lost. This, as discussed in this Instagram post, highlights inefficiency and increases awareness of where time goes.
15-Minute Focused Sprints: Work for 15-20 minutes in a high-focus state. This is often the time required to enter a productive "flow" state, allowing for deep work on tasks like writing, brainstorming, or replying to emails.
The 15-Minute Morning Routine: Start the day without checking your phone for the first 15 minutes. This, mentioned in this Reddit post, avoids chasing "cheap dopamine" and sets a proactive tone for the day.

Key Benefits of 15-Minute Intervals
Overcomes Procrastination: Large tasks are daunting, but 15 minutes is small enough to be "doable".
Increases Awareness: Tracking time in 15-minute blocks reveals how much time is wasted daily.
Maintains Momentum: Short bursts of work can be maintained throughout the day without burnout, allowing for "micro-productivity".
Encourages Deep Work: It provides enough time to focus, but not enough time to be easily distracted if using a timer.
